# PortusSophia™ Canon

## Repository Purpose

This repository contains the **sealed canonical content** of PortusSophia™. All content herein is immutable, foundational, and forms the authoritative doctrine.

## Canonical Content

This repository contains:

- **Postulates**: Immutable foundational statements (`/postulates/`)
- **Doctrine**: Formal doctrinal statements derived from postulates (`/doctrine/`)
- **Mappings**: Authoritative concept and term relationships (`/mappings/`)
- **Hashed Artifacts**: Cryptographically sealed canonical artifacts (`/hashes/`)

## Protection Rules

1. **No alterations** to sealed artifacts, hashes, or doctrine
2. **No drift** from canonical scope
3. **Formal tone** maintained in all documents
4. **Minimal content** approach; only essential canonical material
5. **PortusSophia™ trademark** preserved exactly with ™ symbol

## Non-Canonical Content

The following do **not** belong in this repository:

| Content Type | Redirect To |
|--------------|-------------|
| WebKernel implementations | `portussophia/portussophia-webkernel` |
| Governance templates | `portussophia/portussophia-governance` |
| Narrative content | Appropriate narrative repositories |
| Interpretations | Not accepted |
| Metaphysical expansions | Not within scope |
| Implementation details | Implementation repositories |
